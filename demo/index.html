Autoplay

A jQuery plugin to automate user scenarios

By XY Feng
How it works Source try

Features

Autoplay reads in pre-scripted language line by line, and translate them into 'commands', which controls a simulated cursor moving around screen, typing into input fields, sometime to display messages.

Move cursor around

Cursor can be move to any position on screen, or move to a center of an object in any speed.

Trigger click event

Simulate mouse click event on any given object.

Type into input fields

Simulate type in input or textarea, character by character.

Speed Exit

User still have full control of the mouse cursor, click mouse in everywhere on the page to exit the autoplay mode.

Scroll pages

Scroll page or any given object, supporting both vertical and horizontal direction.

Display message

Display an overlay message and pause the cursor, resume the cursor once the message fade out, fully customizable message layout.

Pause/Resume

Press space bar to pause or resume the running script.

Learning

Write in a special formatted scripting language inside javascript. Each line is composed with command->object->parameters->duration ( NO space before or after -> )
Object is a string, uses jQuery selector, example: #navbar li:first-child a

Wait

wait->3000
let the cursor stay at its location for 3000 milliseconds.

Move

move->#feature_box
move cursor to the center of object #feature_box in default speed.
move->#feature_box->3000
move cursor to the center of object #feature_box in 3000 milliseconds.
move->#feature_box->300,200
move cursor to object #feature_box, offset (300px,200px) from the top-left corner of #feature_box in default speed.
move->#feature_box->300,200->3000
move cursor to object #feature_box, offset (300px,200px) from the top-left corner of #feature_box in 3000 milliseconds.
move->300,200
move cursor to position (300px,200px), offset from the top-left corner of the window in default speed.
move->300,200->3000
move cursor to position (300px,200px), offset from the top-left corner of the window in 3000 milliseconds.

Click

click->#login_btn
trigger click event on object #login_btn

GoClick

goclick->#login_btn
move cursor to the center of object #login_btn and trigger click event

Type

type->#email_input->example@gmail.com
Fill input field #email_input with content example@gmail.com.

GoType

gotype->#email_input->example@gmail.com
Move cursor to the center of input field #email_input and filel with content example@gmail.com character by character.

ScrollX

scrollx->1000
Scroll window horizontally to offset position 1000 pixels.
scrollx->#features_list->1000
Scroll #features_list horizontally to offset position 1000 pixels.

ScrollY

scrollx->1000
Scroll window vertically to offset position 1000 pixels.
scrollx->#features_list->1000
Scroll #features_list vertically to offset position 1000 pixels.

Message

( Default classes: 'center' for an overlay of the whole window; 'bottom' for a overlay bar on bottom of the page. )

message->Scenario 1: Browse to a file.
Display message 'Scenario 1: Browse to a file.' in overlay style for 2000 milliseconds.
message->Scenario 1: Browse to a file.->3000
Display message 'Scenario 1: Browse to a file.' in overlay style for 3000 milliseconds
message->message_class->Scenario 1: Browse to a file.
Display message 'Scenario 1: Browse to a file.' in overlay style for 2000 milliseconds, use .message_class for custom styling.
message->message_class->Scenario 1: Browse to a file.->3000
Display message 'Scenario 1: Browse to a file.' in overlay style for 3000 milliseconds, use .message_class for custom styling.
